# My first Python Project

## Calculator

A simple Python calculator project built to practise Python fundamentals.

### Current Features

- Performs basic arithmetic operations: addition, subtraction, multiplication, and division
- Supports decimal calculations using float numbers
- Validates user input for numbers and operations
- Handles calculation errors such as division by zero

### Current Version

Version 0.6 - Added input and calculation error handling

### Planned Improvements

- Refactor the calculator using functions
